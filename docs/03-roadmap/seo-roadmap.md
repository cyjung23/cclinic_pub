# SEO·콘텐츠 로드맵 (중장기)

> 최종 갱신: 2026-04-11

## 중기 (1~2개월)

- GA4 Key Event 5개 설정 완료 및 데이터 수집 시작
- Paid Search 랜딩페이지 첫 화면 개선, 모바일 로딩속도 최적화
- 블로그 포스팅 빈도 증가 — 소프트주사 소개 콘텐츠 2~3편 발행
- 영문 가격표 이미지 교체 및 영문 사이트 최종 검수
- 한글-영문 동기화 체크리스트 문서화

## 장기 (3개월 이상)

- 유튜브 + 블로그 + 홈페이지 연계 콘텐츠 전략
- GA4 데이터 기반 광고 ROI 분석 및 예산 재배분
- 소프트주사 계열 홈페이지 구축 (한글·영문 동시)
- 영문 사이트 Google Ads 국제 캠페인 검토
