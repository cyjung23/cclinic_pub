# 영문 사이트 (i18n) 구조

> 최종 갱신: 2026-04-11

영문 사이트는 모바일 페이지(m/)를 기반으로 en/ 폴더에 구축. 원페이지 구조로 시술 상세 링크 없이 자체 완결.

## 폴더 구조

    en/
    ├── index.html              (영문 메인, lang="en")
    ├── thank-you/index.html    (영문 상담완료)
    ├── data/
    │   ├── services-index.json
    │   ├── services/           (8개: fatgraft, adhesion, eyelid, faceline, vline, powerline, cellulite, striae)
    │   ├── reviews/            (6개: fatgraft, adhesion, eyelid, faceline, vline, powerline)
    │   └── media/articles.json
    └── js/
        ├── app.js
        ├── service-panel.js
        ├── review-panel.js
        ├── media-panel.js
        └── ga4-events.js

## 번역 범위

서비스 카드 8종, 서비스 상세 패널(How It Works, Benefits, FAQ), 리뷰 6개 카테고리, 언론보도 7건, UI 텍스트(버튼, 안내문구, 폼 메시지), 메타태그(title, description, OG), JSON-LD, thank-you 페이지 전체 영문 전환.

## 기술 참고

- JS가 참조하는 JSON 키: howItWorks(process 아님), benefits, faq, cases, intro, title, badge, icon
- 새 서비스 추가 시: services-index.json + 서비스 JSON + 리뷰 JSON + service-panel.js serviceIdMap 모두 추가
