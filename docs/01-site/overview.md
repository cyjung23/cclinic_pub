## 기술 스택

cclinic.kr은 순수 정적 HTML/CSS/JS로 구축, Cloudflare Pages에서 호스팅됩니다. 서버사이드 렌더링, DB, 프레임워크(Next.js, Supabase 등)는 사용하지 않습니다. 콘텐츠 규모가 50-100페이지 이상이 되거나 주간 단위 업데이트가 필요해질 때까지 현 구조를 유지합니다.
