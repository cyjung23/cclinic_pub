# 작업 가이드라인

> 최종 갱신: 2026-04-11

## 원장님 작업 스타일

전문적인 대화 톤 선호. 마케팅 + 의학적 관점 모두 중시. 데이터 기반 의사결정(키워드 검색량, 트래픽 분석). 블로그 작성 프로세스: 키워드 리서치 → 제목에 키워드 배치 → 참의원 시술 참고 → 경쟁 콘텐츠 대비 차별화.

## 지방파괴주사 핵심 차별점 (모든 콘텐츠 일관 적용)

비스테로이드(트리암시놀론 미사용), 지방세포 파괴(요요 없음, 영구적), 지방 제거 + 타이트닝 동시, 절개·마취·회복기간 불필요, 5~10분 시술 즉시 일상복귀, 좌우 비대칭 교정 가능.

## 소프트주사 핵심 차별점 (신규 콘텐츠 적용)

섬유조직 타겟(지방세포 아님), 과증식 섬유조직 파괴 → 딱딱한 피하조직 연화, 수술 후 흉살·유착·바이오본드 특화, 기존 지방파괴주사와 병행 가능.

## 기술적 주의사항

cclinic.kr은 유료 광고 운영 중 → dev 브랜치 테스트 후 main 병합. Codespaces Live Server로 미리보기 후 push.
